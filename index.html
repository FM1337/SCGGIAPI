SCGGIAPI

SCGGIAPI also known as Steam current game & game inventory API is a python script written by Liam Stanley & Allen Lydiard

The script fetches information from Steam's web API and outputs the information fetched into a text file containing HTML tags.

As well it fetches information from metascore such as the rating and game's description and puts it in a database and updates it every month so it does not have to keep fetching it everytime the script is run.

It fetches the following information from the Steam Web API:

appid

total playtime

app name

profile name

profile url

profile picture

game logo

current game
